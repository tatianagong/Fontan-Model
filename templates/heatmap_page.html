Variable Response Heatmaps

Like the variable response plots, this page shows how variables in our model change when parameters change.

Unlike the plots page, this heatmap page shows how any one variable changes when two parameters change in unison. In a sense, this is a 3D version of the variable response plots feature.

You can select any two different parameters. We omit the 5 compliance value parameters for simplicity, as they are unlikely to be useful. Thus, there are 8 (out of the 13) parameters to chose from. Exact values of each parameter are shown on the x- and y-axes. These values again represent a range from 0.5x to 1.5x the baseline.

You can select one of 6 variables to inspect. The variable value is reflected by color, from blue to red, shown by the sidebar on the right-hand side.
