Variable Response Plots

Our model of the hemodynamics of a Fontan circulatory system solves for 11 variables using 13 parameters.

In this page, you can select any one of the 11 variables, plus the Oxygen Extraction Ratio, to generate a 2D plot. The plot will display how the selected variable changes (on the y-axis) as each of the 13 parameters changes (on the x-axis). The x-axis is normalized to represent each parameter as it goes from 0.5x to 2x its baseline value.

Note: not all variable response plots will show all 13 parameters as not all parameters are relevant for each variable.
